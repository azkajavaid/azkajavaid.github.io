Box plot (ggplot)

Date Thu 15 March 2018 Category Visualization

Import packages

library(ggplot2)


Boxplot of miles per gallon, mpg, against number of forward gears, gear from mtcars dataset

mtcars$gear <- as.character(mtcars$gear) # convert gear to character 
ggplot(mtcars, aes(x = gear, y = mpg, color = gear)) + geom_point(size = 0.7) + geom_boxplot() + ggtitle("Gear vs. miles per gallon") + theme_bw() + theme(text = element_text(size = 20))
